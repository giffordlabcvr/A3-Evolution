APOBEC3-Evolution

Comparative evolutionary analysis of APOBEC3 genes

Download .zip Download .tar.gz View on GitHub

Description

This GitHub repository contains data and program logic supporting comparative and phylogenetic investigations of APOBEC3 evolution.

Sequence data

The sequence data in this project have been organised into the following sources:

ncbi-refseqs: mRNA reference sequences of A3G genes from distinct species. These XML-formatted files are downloaded directly from NCBI using a GLUE module (see here) and are uniquely identified within this project by their GenBank accession numbers.

fasta-curated: A non-redundant set of A3G-like loci. These FASTA sequences have been curated via systematic screening of whole genome sequence (WGS) assemblies using the DIGS tool. Sequences in this source have unique IDs based on arbitrary numbering.

fasta-refseqs: Reference sequences of A3G genes not included in NCBI (e.g. pseudogenes).
