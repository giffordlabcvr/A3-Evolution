A3G-GLUE

A GLUE project for APOBEC3G

Description

This is A3G-GLUE: a GLUE project for APOBEC3G.

GLUE is an open source, data-centric bioinformatics environment specialised for developing virus genome data resources (VGDR).

This GLUE project contains reference information for APOBEC3G (A3G), including:

  1. A set of A3G reference sequences linked to auxiliary data.
  2. A comprehensive list of A3G genome features and their locations within flavivirus reference sequences.
  3. Alignments of A3G reference sequences arranged hierarchically by clade.

Installation

A3G-GLUE can be installed on computers running Windows, MacOSX or Linux, as follows:

  1. Install GLUE, based on the GLUE installation instructions.
  2. Clone the A3G-GLUE repository into your `gluetools/projects` directory.
  3. Navigate to the `gluetools/projects/A3G-GLUE` directory on the command line, initiate GLUE, and build the project by issuing the following command in the GLUE shell:
Mode path: /
GLUE> run file ifnl.glue
